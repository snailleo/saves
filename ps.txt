
1.ctrl + o(字母)  打开对话框
2. ctrl + " 网格（参考线的单位）
   ctrl + k(首选项)  

   ctrl + : 参考线

   ctrl + r (标尺)

3.shift约束键，约束到正方形或圆形
4.alt：以鼠标落点为中心点
5.前景色 ：alt + delete
  背景色： ctrl + delete

6.ctrl + alt + z 撤销操作

7.新建图层： ctrl + shift + alt + n
          或：ctrl + alt + n + enter

8. ctrl + t 调整大小

9. 选区的布尔运算：
   1.新选区      无
   2.加          shift
   3.减          alt
   4.相交        shift + alt

10.羽化快捷键   shift + f6

11.画笔工具：  B;
   笔尖大小；  [小    ]大
   笔尖硬度：shift + [ 软     shift + ] 硬 

12. 合并图层；ctrl + e
    复制图层：ctrl + j  或  alt

13.套索工具：按esc取消之前所有的绘制
   按退格键，取消单步（拔掉木桩）

   按住alt，拖曳则暂时切换到套索工具绘制曲线，
   单击则暂时切换到多边形套索工具绘制直线

14.调整图层顺序：ctrl + [   下
                 ctrl + ]   上
		ctrl + shift [ 到最底层
		ctrl + shift ] 到最顶层

15.图层转换选择选取：按住ctrl 点击图层预览区

16.文字工具：t
     结束文字输入：ctrl + 回车
     选中编辑文字：alt + 左右键   调整文字间距
     选中编辑文字： alt + 上下键   调整行距

17.选取反选：ctrl + shift + i

18.调出图层样式的方法：双击
    复制图层样式：alt + l + y + c
                  alt + l + y + p
    清楚图层样式：alt + l + y + a

19.海尔案例制作步骤：
   1.先设置好前景色和背景色（前：灰 背：白）
   2.在渐变工具属性栏第一项中，选择从前景色到背景色的过度渐变（
     预设中第一项）
   3.选择属性栏中，第二栏第一个（线性渐变）
   4.按住鼠标左键，同时用shift约束垂直拖曳（选取范围内）

20.解决锚点错误的解决方法
   1.删除锚点
     删除当前所选择的锚点，删除之后，要在当前最新的锚点上单击（续上），这样才可以在     接下来的绘制中保持同一条路径的绘制。
   2.移动锚点
     小黑：整体
     小白：细节（路径，锚点，杠杆）
   3.ctrl+alt+z
   4.ctrl + enter 转换为选区
